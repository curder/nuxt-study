## 本地运行

### 环境要求

- Node.js >= 24.0
- Yarn 4.17.1

### 启动

```shell
git clone git@github.com:curder/nuxt-study.git
cd nuxt-study
yarn
yarn docs:dev
```

### 构建

```shell
yarn docs:build
```
